# Initial Thin Slice

The first content slice is the **Dragon / Chaos Waters** theme. Phase 0 only records this as seed metadata; it does not assert unsourced BibleProject claims.

The slice should exercise the whole-canon reader substrate rather than a single-book demo. Passage anchors should be able to point anywhere in the BSB canon, and the data model should allow theme links that move across books and testaments.

Links are directional by default. A relation packet should say which anchor points to which other anchor and why. Backlinks can be generated or displayed later when useful, but they are optional and should not be assumed to carry the same interpretive weight as the reviewed forward link.

The initial source set should stay small enough to review carefully: roughly 5-15 BibleProject sources. The target is 20-50 reviewed packets, enough to test the reader, relation model, duplicate handling, and QC gates without pretending the theme is complete.
